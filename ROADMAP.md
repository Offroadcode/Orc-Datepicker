# Roadmap

## Feature List

* ES6 code
* Single Month
* Ability to black out invalid dates
* Select start date
* Option to select end date or duration
* Navigate forward/backward between months
* Date(s), once selected, populate into linked input field
* Can opt to format the date selected or fired into picker
* KISS

## Does Not Have

* Month selection dropdown for navigating months
* Two month display
* Time selection
* External dependencies
